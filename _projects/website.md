### Tools and credits

I had zero experience in webpage design/development, so I decided to take some time to learn a few things and make this personal website.

The website uses the [**read-only**](https://html5up.net/read-only) template by [**htmlup.net**](https://html5up.net/) and is hosted on [**Github-Pages**](https://pages.github.com/).

I just changed the structure a little to to better modularize it and fit my needs, and not being a huge fan of HTML, this website also takes advantage of [**Jekyll**](https://jekyllrb.com/) to translate my markdown files into static web pages.

Not to be taken as example, but for anyone curious the repository is available on my [**Github profile**](https://github.com/ZhengYiHu/Portfolio).
